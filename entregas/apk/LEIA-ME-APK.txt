O APK Android deve ser gerado a partir da raiz do projeto com:

.\gradlew.bat assembleDebug

Saida esperada:
app\build\outputs\apk\debug\app-debug.apk

Nesta maquina, a geracao do APK foi bloqueada porque o Android SDK nao esta instalado/configurado.
Erro encontrado pelo Gradle:
"SDK location not found. Define a valid SDK location with an ANDROID_HOME environment variable or by setting the sdk.dir path in your project's local properties file"

Como resolver:
1. Instalar Android Studio ou Android SDK Command-line Tools.
2. Confirmar que existe uma pasta como:
   C:\Users\gabri\AppData\Local\Android\Sdk
3. Criar o arquivo local.properties na raiz do projeto com:
   sdk.dir=C\:\\Users\\gabri\\AppData\\Local\\Android\\Sdk
4. Rodar novamente:
   .\gradlew.bat assembleDebug
5. Copiar o arquivo gerado para esta pasta apk.

Observacao:
O caminho do JDK foi ajustado em gradle.properties para o JDK instalado nesta maquina:
C:\Program Files\Microsoft\jdk-17.0.19.10-hotspot
